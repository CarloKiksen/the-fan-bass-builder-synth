# the-fan-bass-builder-synth
A web-based acid bass synthesizer inspired by retro hardware. I built it using Gemini Canvas, to show artists how to built digital artist merch and use AI for ideation. I've captured the process in my newsletter The Fanbase Builder. The sythn is a web-app with a single HTML file using Tone.js for audio and Tailwind CSS for the interface.

- Run the synth in GitHub Pages - https://github.com/CarloKiksen/the-fan-bass-builder-synth
- Read about the idea in my newsletter - https://newsletter.thefanbasebuilder.co/p/i-built-a-web-synth-in-four-hours

## ðŸŽ› Controls
- Generate Pattern: Creates a new random 16-step melody in C Minor.
- Waveform: Switches between Sawtooth (buzzy) and Square (hollow) waves.
- Cutoff: Controls the brightness of the sound.
- Res: Adds that classic "squelch" or whistling sound to the filter.
- Env Mod: Determines how much the filter opens up on each note.
- Drive: Adds distortion and grit.
- Echo / Space: Adds delay and reverb effects.
- Tempo: Controls the speed (BPM).
